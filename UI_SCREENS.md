# UI Screen List

## 1. 온보딩

### 1-1. 스플래시
- 로고 + 슬로건

### 1-2. 캐릭터 소개
- 8캐릭터 카드 슬라이드
- 이름, 한줄 소개
- "시작하기" 버튼

---

## 2. 메인 (홈)

### 2-1. 거실 (Living Room)
- 현재 대화 중인 캐릭터들 썸네일
- 실시간 대화 프리뷰
- "참여하기" / "관전하기" 버튼

### 2-2. 캐릭터 목록
- 8캐릭터 그리드/리스트
- 온라인 상태 표시
- 탭: 전체 / 즐겨찾기

---

## 3. 채팅

### 3-1. 그룹 채팅
- 상단: 참여 캐릭터 아바타들
- 중앙: 채팅 버블 (캐릭터별 색상 구분)
- 하단: 입력창 + 관전모드 토글

### 3-2. 1:1 채팅
- 상단: 캐릭터 프로필 미니
- 중앙: 채팅 버블
- 하단: 입력창

### 3-3. 관전 모드
- 캐릭터들 대화 자동 진행
- "끼어들기" 플로팅 버튼
- 배속 조절 (1x, 1.5x, 2x)

---

## 4. 캐릭터 상세

### 4-1. 프로필
- 캐릭터 이미지 (큰 사이즈)
- 기본 정보 (이름, 나이, MBTI, 직업)
- 성격, 좋아하는 것/싫어하는 것

### 4-2. 관계도
- 해당 캐릭터 중심 관계망 시각화
- 관계 타입 아이콘 (친구, 호감, 라이벌 등)

### 4-3. 대화 기록
- 과거 대화 히스토리
- 날짜별 그룹핑

---

## 5. 상황 설정 (연출 모드)

### 5-1. 장소 선택
- 카페 / 공원 / 학교 / 바 / 집 등
- 장소별 썸네일

### 5-2. 캐릭터 선택
- 참여 캐릭터 선택 (2~4명)
- 드래그로 배치

### 5-3. 상황 설정
- 프리셋: "우연히 마주침" / "생일파티" / "고민상담" 등
- 커스텀 상황 입력

---

## 6. 설정

### 6-1. 앱 설정
- 알림 on/off
- 다크모드
- 언어

### 6-2. 계정
- 로그인/로그아웃
- 데이터 초기화

---

## 화면 흐름 요약

```
스플래시 → 온보딩 → 메인(거실)
                      ↓
        ┌─────────────┼─────────────┐
        ↓             ↓             ↓
    그룹채팅      1:1채팅      캐릭터상세
        ↓
    관전모드
        ↓
    상황설정 → 새 채팅방 생성
```
