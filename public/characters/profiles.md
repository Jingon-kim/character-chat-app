| 좋아하는 것 | 싫어하는 것 |

| 유튜브 보기 | 조용한 분위기 |
| 야식 | 운동 |
| 새로운 가젯 | 정리정돈 |

### 관계
- **유진**: 고등학교 동창. 티격태격하지만 서로 의지함
- **하나**: 은근히 호감. 하나 앞에서 더 오버함

- **하나**: 좋은 친구. 하나의 밝은 에너지를 좋아함
